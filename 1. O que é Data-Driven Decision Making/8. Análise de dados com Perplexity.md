# Introdução
A aula apresenta o **Perplexity** como a terceira ferramenta de IA online para análise de dados. Ainda não tão popular quanto as anteriores, ele se destaca pela capacidade de **pesquisar tendências de mercado em tempo real**, complementando dados internos com fontes verificáveis, ideal para benchmarks e validações externas.

# Conteúdo Principal

## Destaques do Perplexity para análise de dados
- **Pesquisa de tendências de mercado**:
  - Complementa **dados internos** com informações externas públicas.
  - Útil para **benchmarks**: Comparar rotinas de entregas de dev, desempenho de times ou métricas de negócio com dados de outras empresas.
- **Fontes verificáveis**:
  - Traz **referências confiáveis** em relatórios gerados.
  - Permite **verificação manual**, evitando "alucinações" — baseia-se em uma **knowledge base real** pesquisada.
- **Validação de hipóteses de negócio**:
  - Integra dados externos para análises comparativas.
  - Focado em **mensurar performance**: Rotinas, entregas e resultados do time ou produto.
- **Ideal para situações comparativas**:
  - Enriquecimento de relatórios com dados de mercado.
  - Ênfase na **confiabilidade dos dados**.

## Exemplo prático de prompt
> **"Compare os nossos dados de Churn com a média do setor de Telecom de 2025."**

- Aplicável a setores como telecomunicações.
- O Perplexity pesquisa dados públicos, gera comparação e inclui fontes verificadas.
- Resultado: Análise difundida e confiável para decisões estratégicas.

# Conclusão
O **Perplexity** eleva a análise de dados ao conectar o interno com o externo, fornecendo benchmarks rápidos e fontes verificáveis que garantem credibilidade. Perfeito para validar hipóteses de negócio e medir performance contra o mercado, ele reforça o DDDM com evidências reais, tornando-o indispensável para profissionais que precisam de comparações setoriais confiáveis e acionáveis.
