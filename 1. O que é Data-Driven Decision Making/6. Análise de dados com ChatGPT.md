# Introdução
A aula apresenta o **ChatGPT** como a primeira ferramenta de IA online para análise de dados, destacando seu papel como um "queridinho famoso" e versátil. O foco é mostrar como ele auxilia desde desenvolvedores juniores até analistas, gerando códigos, explicando conceitos, visualizando dados e respondendo perguntas de negócio com prompts simples.

# Conteúdo Principal

## Funcionalidades do ChatGPT para análise de dados
- **Geração de código**:
  - Python + SQL para explorar datasets.
  - Machine Learning (avançado): criação de **dados sintéticos** para treinar modelos.
- **Suporte a desenvolvedores**:
  - Ajuda **devs juniores** a estruturar códigos.
  - Útil ao aprender novas linguagens ou lidar com dúvidas técnicas.
- **Explicação de estatísticas e métricas**:
  - Desmistifica conceitos complexos com linguagem **simples e acessível**.
  - Ensina cálculos estatísticos passo a passo.
- **Visualização de dados**:
  - Cria **gráficos** a partir de JSON, CSV, etc.
  - Gera **tabelas prontas** para análise visual.
  - Ideal para volumes **pequenos a médios** (grandes datasets podem sobrecarregar o contexto).
- **Análise de negócio**:
  - Responde perguntas com base em métricas fornecidas.
  - Realiza **correlações simples** e identifica padrões.

## Exemplo prático de prompt
> **"Leia esse CSV de vendas e mostre os top 5 produtos mais vendidos em 2024."**

- Simples, direto e funcional.
- O ChatGPT processa o arquivo, ordena os dados e entrega o resultado rapidamente.
- Economiza tempo em comparação à análise manual.

## Preparação para Prompt Engineering
- Os prompts apresentados são **básicos**, para iniciar a interação.
- Futuramente, o curso abordará **técnicas avançadas de Prompt Engineering** para extrair mais valor da IA.

# Conclusão
O ChatGPT é uma ferramenta poderosa e acessível para análise de dados, funcionando como um **assistente completo**: gera código, explica estatísticas, cria visualizações e responde perguntas de negócio com agilidade. Especialmente útil para devs juniores e profissionais sem tempo para análises manuais, ele democratiza o DDDM ao transformar prompts simples em insights acionáveis, preparando o terreno para práticas mais avançadas com IA.
