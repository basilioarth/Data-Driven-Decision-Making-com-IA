# RELATÓRIO EXECUTIVO — CHURN DA PLATAFORMA  
(100% baseado na análise final do ChatGPT + correções manuais que fizemos juntos)

### 1. Taxa geral de churn  
**32,4 %** dos clientes cancelaram no último mês  
(≈ 324 cancelamentos em ~1.000 clientes)

### 2. Os 4 preços reais — o grande vilão ficou óbvio

| Plano + Preço      | % dos clientes | Taxa de churn | Observação                                 |
|---------------------|----------------|---------------|--------------------------------------------|
| Basic 49,90         | 38 %           | 26 %          | Mais fiel                                  |
| Premium 49,90       | 12 %           | 31 %          | Ok                                         |
| Premium 99,90       | 23 %           | 36 %          | Alto churn                                 |
| **Basic 99,90**     | **27 %**       | **42 %**      | **PIOR SEGMENTO — 42 % churn!**            |

**Insight bomba**: o **Basic 99,90** é o plano que mais cancela — paga caro por algo que deveria ser “básico”.

### 3. Motivos declarados (ChurnReason) — ordenado corretamente

| Motivo               | % do churn total | Comentário rápido                                 |
|----------------------|------------------|---------------------------------------------------|
| **Cost**             | **41 %**         | Vilão absoluto                                    |
| Difficulty of Use    | 19 %             | UX/UI ruim — time de Produto                     |
| Bug                  | 16 %             | Erros críticos — time de Dev                     |
| Accessibility        | 9 %              | WCAG / leitor de tela — obrigatório              |
| Other                | 9 %              | Imprevisível                                      |
| Competition          | 6 %              | Menor preocupação agora                           |
| None                 | (ignorar — são os ativos) | —                                        |

### 4. Top 5 combinações letais (as que precisamos atacar já)

| Combinação                                      | Taxa de churn | % dos cancelamentos |
|-------------------------------------------------|---------------|---------------------|
| Basic 99,90                                     | 42 %          | 27 %                |
| Premium 99,90 + satisfação ≤ 2                  | 68 %          | 11 %                |
| Qualquer plano + ≥ 3 tickets de suporte         | 59 %          | 9 %                 |
| Qualquer plano + ≤ 5 logins no mês              | 51 %          | 8 %                 |
| Satisfação = 1 ou 2 (qualquer preço)            | 77 %          | 14 %                |

### 5. 3 AÇÕES CONCRETAS PARA APRESENTAR NA REUNIÃO DE AMANHÃ

| Owner                     | Ação (prazo máximo 30 dias)                                    | Impacto esperado         |
|---------------------------|----------------------------------------------------------------|--------------------------|
| **Dev + Produto**         | Revisar e simplificar fluxos críticos + corrigir top 10 bugs mais reportados | –19 % Difficulty/Bug     |
| **Suporte + Sucesso**     | Programa de retenção ativo para quem tem ≥ 3 tickets ou satisfação ≤ 2 → ligar/offerta desconto | –12 % churn de alto risco|
| **Marketing + Preço**     | (1) Renomear ou extinguir o “Basic 99,90”  <br>(2) Criar plano intermediário 79,90 com features claras | –15–20 % churn por custo |
