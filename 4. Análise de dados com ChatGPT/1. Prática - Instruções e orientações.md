# AULA PRÁTICA: Análise de Churn com IA (Mão na Massa!)

**Objetivo da aula**  
Transformar uma planilha de **churn de clientes** (100% sintética e segura) em um **relatório executivo completo** usando **3 IAs diferentes**:

1. **ChatGPT** (análise clássica + visualizações)  
2. **Manus AI** (gráficos interativos, heatmaps, sankey)  
3. **Lovable** (dashboard prototipado em minutos)

**Arquivo da aula** → **churn_plataforma_sintetico.csv** (já disponível no material do curso – 100% seguro, sem PII)

### Estrutura da Planilha (Principais colunas)

| Coluna                  | Descrição                                      |
|-------------------------|------------------------------------------------|
| cliente_id              | ID fictício do cliente                         |
| idade                   | Idade                                          |
| regiao                  | Sudeste, Nordeste, Norte, Sul, Centro-Oeste   |
| plano                   | Basic, Pro, Enterprise                         |
| valor_mensal            | Valor pago por mês                             |
| meses_ativo             | Tempo de assinatura                            |
| ticket_suporte          | Quantidade de tickets abertos                  |
| nps                     | Nota NPS (0–10)                                |
| ultimo_login_dias       | Dias desde o último login                      |
| churn                   | 1 = cancelou, 0 = ativo                        |
| motivo_cancelamento     | Texto livre (quando aplicável)                 |
