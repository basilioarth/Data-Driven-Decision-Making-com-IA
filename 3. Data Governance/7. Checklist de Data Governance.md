## CHECKLIST: 5 Perguntas ANTES de Enviar Dados para IA

| # | Pergunta | Se SIM → | Se NÃO → |
|---|---------|----------|----------|
| 1 | **Os dados são confidenciais?** | → Vá para 2 | → **SEGURO** (mas verifique leis) |
| 2 | **Os dados foram anonimizados?** (tokenização, mascaramento, agregação) | → Vá para 3 | → **PARE. ANONIMIZE AGORA.** |
| 3 | **Existe risco de vazamento mesmo anonimizado?** | → **NÃO ENVIE** | → Vá para 4 |
| 4 | **Viola política interna ou lei?** (LGPD, GDPR, HIPAA, etc.) | → **NÃO ENVIE** | → Vá para 5 |
| 5 | **Posso criar dados fictícios/sintéticos?** | → **USE DADOS FICTÍCIOS** | → **USE IA LOCAL / SANDBOX** |

> **Se qualquer resposta for "SIM" no caminho errado → NÃO ENVIE.**

---

## EXEMPLOS PRÁTICOS

| Dados | Confidencial? | Anonimizado? | Risco? | Viola lei? | Ação |
|------|---------------|--------------|-------|------------|------|
| `nome, cpf, venda` | ✅ Sim | ❌ Não | ✅ Sim | ✅ LGPD | → **ANONIMIZE ou crie sintético** |
| `total_vendas_SP` | ❌ Não | ✅ (agregado) | ❌ Não | ❌ Não | → **SEGURO** |
| `email, cartão` | ✅ Sim | ✅ (mascarado) | ❌ Não | ❌ Não | → **SEGURO (se DPA existe)** |

## COMO ANONIMIZAR (Rápido e Seguro)

```python
# 1. Tokenização
df['cliente_id'] = range(1, len(df)+1)
df = df.drop(columns=['nome', 'cpf', 'email'])

# 2. Mascaramento
df['email'] = df['email'].str[0] + '***' + '@' + df['email'].str.split('@')[1]
df['cpf'] = '***.***.' + df['cpf'].str[-6:]

# 3. Agregação
df_agg = df.groupby('regiao')['valor'].mean().reset_index()
```

## DICAS FINAIS
