# Resumo da Aula – Primeira Tentativa com Lovable AI

## Objetivo
Criar um **dashboard interativo** (100 % clicável, bonito e compartilhável) usando os mesmos dados de churn, mas agora via **Lovable** (Vibe Coding).

## Estratégia usada
1. Pedir ao ChatGPT um prompt otimizado para Lovable  
2. Colar o prompt + anexar a planilha  
3. Deixar público (versão free) e rodar

## O que aconteceu (em menos de 10 minutos)
- Lovable leu a planilha e entendeu as colunas  
- Criou um dashboard completo com:
  - Overview (métricas principais)
  - Gráficos interativos (churn por região, plano, motivo, etc.)
  - Seções de insights e recomendações
  - Design limpo e moderno

## Problemas identificados na primeira versão
| Item                     | Resultado da Lovable              | Realidade correta |
|--------------------------|-----------------------------------|-------------------|
| Total de clientes        | 10                                | 300               |
| Clientes ativos          | 7                                 | 204               |
| Taxa de churn            | ~30 % (mostrou errado no card)    | 32 %              |
| Análise dos motivos      | Confusa / incompleta              | Custo é o principal |

**Conclusão do professor:**  
Lovable tentou fazer a **análise + visualização** ao mesmo tempo → acabou errando os números básicos.  
Ela é ótima em **design e interatividade**, mas **não é tão forte em análise estatística pesada** quanto a Manus AI.

## Lição tirada (e próxima ação)
**NÃO pedir para Lovable fazer a análise do zero.**  
O fluxo ideal é:

1. Fazer a análise profunda com **Manus AI** (ou Python/ChatGPT) → já temos o relatório perfeito  
2. Copiar os **insights prontos** e colar como texto no prompt da Lovable  
3. Pedir apenas: “Crie um dashboard interativo lindo com esses insights e essas métricas exatas que eu vou te passar”

Ou seja:  
**Manus AI = cérebro analítico**  
**Lovable = designer + front-end instantâneo**

Próximo passo da aula: refazer o prompt da Lovable **sem pedir análise**, só pedindo visualização bonita dos resultados que já sabemos que são corretos. O dashboard vai ficar perfeito!
