# Resumo da Aula – Data Driven Decision Making com IA (continuação)

## Objetivo da aula
Continuar o desenvolvimento de uma análise de dados de **churn de clientes** utilizando a **Manus AI**, melhorando o prompt anterior para obter um relatório mais completo, preciso e acionável.

## Problemas identificados no prompt anterior
- Foco excessivo em análise descritiva simples e correlações básicas.
- Risco de a Manus AI perder formatação ou não entregar todos os elementos desejados (insights profundos, gráficos, recomendações).
- Uso de prompts muito setorizados (vários prompts separados) pode sobrecarregar a Manus AI, especialmente com arquivos que, embora não sejam big data, exigem processamento coordenado.

## Estratégia adotada
1. **Usar o ChatGPT como “prompt engineer”** para criar um prompt único e otimizado especificamente para a Manus AI.
2. Deixar o prompt mais genérico, mas ainda bem direcionado, evitando divisões excessivas.
3. Manter a restrição importante: a Manus AI deve atuar como **analista quantitativa sênior**, focada em:
   - Resultados numéricos
   - Gráficos claros
   - Insights práticos e acionáveis
   - Recomendações estratégicas para redução de churn
   - **Sem desenvolver modelos preditivos complexos**

## Papel do ChatGPT na aula
- Recebeu o contexto completo (planilha de churn, objetivo de relatório com insights e recomendações).
- Foi solicitado um **prompt único** adaptado ao estilo que a Manus AI interpreta melhor.
- Entregou um prompt longo, porém completo, que já considera a estrutura ideal de resposta da Manus (tabelas, gráficos, seções claras de insights e recomendações).

## Vantagens do prompt único para Manus AI
- Reduz o risco de perda de contexto ou formatação durante o processamento.
- Simplifica o fluxo: basta colar o prompt + anexar a planilha.
- Aproveita o fato de a Manus AI ser otimizada para relatórios analíticos bem estruturados quando recebe instruções claras e consolidadas.

## Lições práticas destacadas pelo professor
- Recorrer a uma IA (como o ChatGPT) para construir prompts otimizados para outra IA (Manus AI) é uma técnica poderosa e recomendada.
- Cada ferramenta de IA tem “nuances” de interpretação; adaptar o prompt ao “estilo” da IA alvo melhora significativamente o resultado.
- Mesmo sem ser um especialista em prompt engineering, é possível obter prompts de alta qualidade delegando essa tarefa ao próprio ChatGPT, desde que o contexto seja bem passado.
