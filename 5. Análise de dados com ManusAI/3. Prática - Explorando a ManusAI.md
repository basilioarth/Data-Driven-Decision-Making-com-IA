# Resumo da Aula – Explorando a Plataforma Manus AI (versão gratuita)

## Visão geral da interface
- Interface bem simples e limpa (foto do usuário no canto).
- Versão gratuita: começa com ~1.300 créditos; repõe diariamente (geralmente à meia-noite).
- Existe versão Pro com mais créditos e funcionalidades avançadas.

## Modos de execução disponíveis
- **Adaptativo** → respostas instantâneas, mais rápido, mas pode alucinar ou perder formatação.
- **ADG** → mais autonomia para tarefas complexas.
- **Chat** → ideal para conversas e tarefas simples do dia a dia.

## Opções de saída (output)
- Imagens, Slides, Webpage, Spreadsheet, Visualization, Report, Document, PDF, Code etc.
- Botão “More” → acesso a Playbooks prontos (ex.: construtor de site, gerador de vídeo, análise SWOT, pesquisa de marketing, tradutor de PDF etc.).

## Recurso que mais se destaca na Manus AI
Principal fama da ferramenta: **processamento de dados + geração automática de relatórios completos com gráficos**.  
É exatamente para isso que a maioria das pessoas usa (e ama) a Manus.

## Seção Data / Visualization
- Exemplos prontos de prompts (em inglês, mas funciona com português também).
- Permite fazer upload da planilha e escolher o tipo de visualização desejada:
  - Heatmap (favorito do professor)
  - Sankey Graph
  - Scatter Plot
  - Radar Chart
  - Treemap  
  …e vários outros.
- Dá para selecionar até 5 tipos de gráficos diferentes de uma vez.

## Opções finais de entrega
Ao final você escolhe o formato do resultado:
- Report (relatório completo) ← o que vamos usar
- Slides
- Webpage
- Spreadsheet
- etc.

## Decisão tomada na aula
Mesmo sendo possível usar o modo “Visualization” ou simplesmente subir a planilha + clicar em “Report”, o professor optou por **não usar esses atalhos**.  
Vamos colar o prompt longo e robusto (aquele criado com ajuda do ChatGPT) diretamente no campo de texto principal, sem pré-selecionar visualizações específicas, para testar todo o poder da Manus AI com instruções detalhadas em português.

**Próximo passo real da aula:**  
Anexar a planilha de churn + colar o prompt completo + executar e ver o relatório executivo que a Manus vai gerar do zero.
