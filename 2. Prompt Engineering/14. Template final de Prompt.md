# Receita do Bolo do Prompt  
*(Template Oficial de Prompt Engineering – Salve, marque, use sempre!)*

---

## **1. Papel da IA**  
> *Quem ela é?*  
```plaintext
Você é um [PROFISSIONAL] com [X anos] de experiência em [ESPECIALIDADE].
```

---

## **2. Contexto**  
> *Onde estamos? Qual o cenário?*  
```plaintext
Estamos analisando [DADO/OBJETIVO] para [FINALIDADE].
Ex: vendas de 2024 para identificar a região com menor acesso ao site.
```

---

## **3. Tarefa Específica**  
> *O que ela deve fazer?*  
```plaintext
[VERBO DE AÇÃO] + [OBJETO]  
Ex: Filtre vendas acima de R$1.000 / Gere um resumo estratégico / Crie um dashboard
```

---

## **4. Formato de Saída**  
> *Como entregar?*  
```plaintext
Retorne em [FORMATO]  
Ex: tabela Markdown / bullet points / JSON válido / PNG / CSV / código Python
```

---

## **5. Tom e Estilo**  
> *Como falar?*  
```plaintext
Use linguagem [TIPO] para [PÚBLICO]  
Ex: clara e objetiva para stakeholders / técnica para devs / como se eu fosse uma criança
```

---

## **6. Restrições**  
> *O que NÃO fazer?*  
```plaintext
- Máximo [X] palavras/linhas  
- Ignore [COLUNA/DADO]  
- Não use [PALAVRA/TECNOLOGIA]  
- Evite jargões
```

---

# Exemplo Completo (Lovable – Automação de Dados)

> ```
> Você é um **engenheiro Python focado em automação de dados**.  
>  
> Estamos processando o arquivo `vendas.json` com registros de vendas de 2024.  
>  
> **Crie um script Python** que:  
> 1. Leia o JSON  
> 2. Filtre vendas com `valor > 1000`  
> 3. Gere um gráfico de barras  
>  
> **Salve como `vendas_acima_1000.png`**.  
> Use **comentários em cada etapa**.  
> Linguagem **técnica, clara para juniores**.  
> **Máximo 50 linhas de código**.
> ```

# Dica de Ouro (da Professora)

> **Não precisa usar os 6 passos sempre.**  
> **Os 4 obrigatórios são:**  
> 1. **Contexto**  
> 2. **Tarefa**  
> 3. **Formato**  
> 4. **Restrição**  

> Com esses 4, você já tem **90% do poder** de um prompt avançado.
